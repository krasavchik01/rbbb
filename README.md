## 🚀 Быстрый старт

### Простой способ (Windows):

1. **Запустите `START.bat`** - двойной клик по файлу
2. **Откройте браузер:** `http://localhost:8080`
3. **Войдите используя демо-пользователей** (см. ниже)

### Универсальный способ:

```bash
# 1. Установите зависимости (только первый раз)
npm install

# 2. Запустите сервер разработки
npm run dev

# 3. Откройте http://localhost:8080
```

## 🛠 Технологии

- **Frontend:** React 18, TypeScript, Vite
- **UI:** Shadcn/UI, Tailwind CSS

- **State:** React Query, Context API

## 🐛 Решение проблем

### Белый экран после запуска:

1. **Жесткое обновление браузера:** `Ctrl + Shift + R`
2. **Очистка кеша в консоли (F12):**
   ```javascript
   localStorage.clear(); 
   sessionStorage.clear(); 
   location.reload(true);
   ```
3. **Или используйте режим Инкогнито**

### Сервер не запускается:

1. Убедитесь что находитесь в правильной папке проекта
2. Проверьте что установлен Node.js: `node -v`
3. Переустановите зависимости: `npm install`
4. Удалите кеш: `npm cache clean --force`

### Порт 8080 занят:

```bash
# Windows (PowerShell):
netstat -ano | findstr :8080
taskkill /PID <PID> /F

# Linux/Mac:
lsof -ti:8080 | xargs kill
```
